Cult Petrópolis

O projeto consiste em um pagina dedicada à cidade de Petrópolis, RJ. Apresenta informações climáticas atualizadas em tempo real,o JavaScript implementa interatividade, permitindo que os dados climáticos sejam atualizados dinamicamente.

Para ter essa função foi usado a biblioteca Axios para realizar requisições HTTP de forma simplificada e para acessar a API do OpenWeather e o CDN: Utilização de bibliotecas hospedadas em servidores externos para facilitar o acesso e otimizar o carregamento do site.

A pagina faz uso da API OpenWeather para obter dados meteorológicos, isso permite que os usuários visualizem a temperatura atual e as condições climáticas de Petrópolis.
